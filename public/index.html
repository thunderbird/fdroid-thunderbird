Thunderbird F-Droid Repository

This F-Droid repository contains Android apps made available by Thunderbird.

How to

Go to F-Droid and download the F-Droid client.

To add this repository to your F-Droid client, please follow these instructions using the provided QR code (click it to enlarge) or the following URL:

https://thunderbird.github.io/fdroid-thunderbird/repo
